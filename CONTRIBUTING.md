# 如何贡献

## 开发流程

1. Fork 本仓库
2. 创建功能分支：`git checkout -b feature/xxx`
3. 提交更改：`git commit -m "feat: 添加 xxx"`
4. 推送分支：`git push origin feature/xxx`
5. 提交 Pull Request

## 提交规范

使用 [Conventional Commits](https://www.conventionalcommits.org/)：

```
feat: 新功能
fix: 修复 bug
docs: 文档变更
style: 代码格式（不影响逻辑）
refactor: 重构
test: 测试
chore: 构建/工具变更
```

## 分支规范

- `main` — 生产分支
- `dev` — 开发分支
- `feature/*` — 功能分支
- `fix/*` — 修复分支
